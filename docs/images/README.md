# Documentation Images

Place your screenshots and images in this folder.

Recommended images:

- `dashboard-preview.png` - Main dashboard screenshot
- `login-page.png` - Login page
- `import-csv.png` - CSV import feature
- `manual-entry.png` - Manual entry form

## Taking Screenshots

For consistent screenshots:

1. Use a browser window at 1280x720 resolution
2. Use light mode (or include both light/dark)
3. Use sample data that looks realistic
4. Avoid showing real personal data
